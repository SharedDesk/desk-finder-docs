<Note>
    Sentry ist ein Open-Source-Tool zur Fehlerüberwachung, das Ihnen hilft, Probleme in Echtzeit zu erkennen und zu
    beheben. Sentry kann in Shared Desk integriert werden, um Fehler in der Anwendung zu überwachen und
    Benachrichtigungen zu erhalten, wenn ein Fehler auftritt.
</Note>

## Schritte zur Integration von Sentry

1. Erstellen Sie ein Sentry-Konto, wenn Sie noch keines haben. Gehen Sie zu [sentry.io](https://sentry.io) und melden Sie sich an.
2. Erstellen Sie ein neues Projekt in Sentry.
3. Kopieren Sie den DSN (Data Source Name) für das Projekt.
4. Fügen Sie den DSN in die Umgebungsvariablen ein:
```bash
SENTRY_DSN=your-dsn
SENTRY_ENVIRONMENT=production
SENTRY_TRACES_SAMPLE_RATE=0.1
SENTRY_SAMPLE_RATE=0.2
```
Für weitere Informationen zu den Sentry Einstellungen siehe [Sentry Dokumentation](https://docs.sentry.io/platforms/java/guides/spring-boot/).
5. Starten Sie die Anwendung neu.
