Alternativ kann auch eine Swagger UI zur Erkundung genutzt werden. Die Swagger UI kann unter [production.shared-desk.cloud/swagger-ui/](https://production.shared-desk.cloud/swagger-ui/index.html) aufgerufen werden.
Die Open API Spezifikation kann unter [production.shared-desk.cloud/api-docs](https://production.shared-desk.cloud/api-docs) abgerufen werden.

<Note>
    Fast alle Endpunkte in Shared Desk sind durch JWT (JSON Web Token) geschützt. Das bedeutet, dass ein gültiges JWT im
    Header der Anfrage enthalten sein muss, um auf die Endpunkte zuzugreifen. Wenn kein JWT vorhanden ist oder das JWT
    ungültig ist, wird der Zugriff verweigert.
</Note>
